# Profile Validation

This folder contains documentation for profile field validation and completeness checking for the Know Me Agent.

## Contents

- **TPS-Field-Categories.md** - Categorized list of Travel Profile Service fields by booking requirement (Required, Recommended, Optional)

## Overview

Profile validation ensures travelers have all necessary information before booking, preventing booking failures and improving user experience by proactively prompting for missing fields.
