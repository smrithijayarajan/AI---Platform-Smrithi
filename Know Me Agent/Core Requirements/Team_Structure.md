## Collaboration Model

The Know Me Agent team operates in a collaborative, cross-functional model where:

- **PM** defines the "what" and "why" - user needs, business value, and success metrics
- **Architect** defines the "how" - technical approach, system design, and implementation strategy
- **UX** defines the "experience" - user flows, interface design, and interaction patterns

All three roles work together throughout the product lifecycle from discovery to delivery.
